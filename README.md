# 📊 Data Visualization using Python

## 🔍 Problem Statement

Visualize startup data to uncover patterns, relationships, and trends using graphical techniques.

## 📁 Dataset

* Startup Growth & Funding Dataset
* Source: Kaggle

## 🛠 Tools & Technologies

* Python
* Pandas
* Matplotlib

## 📊 Analysis Performed

* Data loading and preprocessing
* Visualization using multiple chart types:

  * Pie charts
  * Bar charts
  * Histograms
  * Line charts
  * Scatter plots

## 📈 Key Insights

* EdTech and FinTech dominate industry distribution
* Majority of startups remain private
* Strong positive relationship between funding and valuation
* Revenue increases with employee count

## 💡 Business Impact

* Helps stakeholders understand market trends visually
* Enables quick decision-making through graphical insights
* Improves data storytelling for business presentations

## 📸 Visualizations

* Industry distribution pie chart
* Exit status analysis
* Funding vs valuation graphs

## 🚀 Conclusion

This project demonstrates the power of Python visualization libraries in transforming raw data into meaningful business insights.
